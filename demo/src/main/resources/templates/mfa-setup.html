Registra il tuo dispositivo

Scansiona il QR code con Google Authenticator o Microsoft Authenticator. Dopo la scansione, passerai alla verifica del codice OTP.

Per sicurezza, questo QR code serve solo per configurare il dispositivo. Non condividerlo e non salvarlo in screenshot non protetti.

Il server verificherà la registrazione solo dopo l’inserimento di un codice OTP valido.
